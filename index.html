Objetivos

Por medio del presente OVI se darán a conocer algunas temáticas importantes y necesarias para el desarrollo de sitios web y tiene los siguientes objetivos:

1. Conocer un poco sobre la historia de los sitios web.
2. Conocer que es IDE y como se utiliza para el desarrollo de sitios web.
3. Conocer que es GIT y GITHUB y cuál es su importancia para el desarrollo de los sitios web en la actualidad.
4. Aprender que es HTML, para que sirve, cuál es su importancia en los sitios web y conocer algunas de sus etiquetas y estructura para el diseño de sitios web.
5. Aprender que es CSS, para que sirve, cuál es su importancia en los sitios web y conocer algunas de sus etiquetas y estructura para el diseño de sitios web.

Estos son los contenidos que se van a encontrar en el OVI, mediante algunas lecturas, archivos multimedia y actividades el objetivo es que se pueda conocer y aprender cómo se construyen los sitios web y aplicar los conocimiento adquiridos con este OVI y de esta manera le pueda servir para aplicarlos en su vida académica o tal vez laboral.
